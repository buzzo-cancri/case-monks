# Case Técnico da Monks

Esse repositório se trata do Case Técnico proposto pela Monks Brasil. O objetivo era de estruturar uma página web utilizando de HTML, CSS e JavaScript, seguindo um Mockup de referência. 
Para o desenvolvimento, optei por utilizar o framework para CSS: **Boostrap 5.2**, via CDN. O Bootstrap, além de fornecer funcionalidades que aceleram a estilização da página, também fornece alguns componentes (com suas interações já implementadas) que facilitam o desenvolvimento, como o menu hambuguer. No entanto, foi necessário modificar algumas das classes do Bootstrap e adicionar novos IDs para personalização que se adequasse ao Mockup.
Como os componentes fornecidos pelo Bootstrap são fornecidos via web, não é necessário fazer nenhuma instalação prévia para visualizar a página.
